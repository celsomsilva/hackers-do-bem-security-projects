# Network Scan Lab

Objetivo:
Aprender a identificar dispositivos e serviços em uma rede usando ferramentas de varredura.

Ferramenta sugerida:
nmap

Desafio:

1. escanear uma rede local
2. identificar portas abertas
3. descobrir serviços rodando

Perguntas:

- quais portas estão abertas?
- quais serviços estão rodando?
- quais podem representar risco?
